Тестовое задание с проверкой API авторизации (получение токена).
Содержит следующие файлы:
 - authorization.feature - содержит текстовые сценарии для проверки.
 - __init__.py - базовые реализации шагов для http-запросов.
 - authorization_api.py - реализации шагов для API авторизации (задание доп. параметров и проверка результата).
